# Chatty — Requirements Specification (Release 2)

## Контекст
Chatty — чат-приложение.  
В рамках проекта требовалось подготовить спецификацию требований для второго релиза с учётом развития функциональности.

## Цель проекта
Подготовить структурированную и согласованную спецификацию требований для следующего релиза продукта.

## Роль
Системный аналитик

## Задачи аналитика
- Анализ исходных требований и бизнес-логики
- Формирование функциональных и нефункциональных требований
- Описание сценариев использования
- Структурирование требований для передачи в разработку
- Подготовка спецификации требований

## Артефакты
- Спецификация требований (SRS)
- Use Cases
- Описание функциональности релиза

## Результат
Подготовлена спецификация требований ко второму релизу Chatty.  
Работа прошла ревью и была принята без критических замечаний.

## Статус
✔️ Задание принято, ревью пройдено
